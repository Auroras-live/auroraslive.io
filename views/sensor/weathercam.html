Weathercam

The sample weathercam software lets you turn a Raspberry Pi into a simple cloud cover camera, complete with weather overlay

Getting Started

For this project you'll need an internet connected Raspberry Pi and camera module (in addition to everything else you'd need to use the Pi, such as SD card and power supply). Note that this is intended for people who are comfortable with setting up Raspberry Pi's and using the command line in Linux

Once you've got your Pi ready to go, head over to the weathercam GitHub repo and follow the instructions there
