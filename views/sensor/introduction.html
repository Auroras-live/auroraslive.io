Introduction

The Auroras.live Sensor Network is a group of devices that share information back to the Auroras.live API. Other people can then access this information via the API or an interactive map.

Examples of devices could include:

  • DIY magnetometers
  • Cloud cover or aurora cameras
  • Internet connected weather stations

The DIY nature of the network means anyone can build a device using any hardware / software combination and safely share the information their device logs

To get started, check out the weathercam page which includes sample software for the Raspberry Pi that lets you set up an inexpensive cloud cover camera
