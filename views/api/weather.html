The Weather Module

The weather module retrieves weather for a given location

Weather data is provided by met.no

Required Parameters

Parameter Type Valid ranges Notes
lat Float -90 to 90 The latitude of the area you wish to obtain information for.
long Float -180 to 180 The longitude of the area you wish to obtain information for.

Optional Parameters

Parameter Type Valid ranges Notes
forecast Boolean true / false If set to true, obtains a 9 day weather forecast, split up into 3 hour intervals

Example 1

http://api.auroras.live/v1/?type=weather&lat=40.7813913&long=-73.976902

Returns

Current weather conditions for Manhattan, New York

See Example 

{{ $root.resultsData['weather'] }}

Example 2

http://api.auroras.live/v1/?type=weather&lat=40.7813913&long=-73.976902&forecast=true

Returns

9 day weather forecast for Manhattan, New York

See Example 

{{ $root.resultsData['weatherforecast'] }}
