The ACE Module

The ACE module (named due to the majority of the data coming from the ACE spacecraft) is the core the Auroras.live suite of apps and services. It returns important aurora hunting information, including Bz, Density, Speed, Kp forecasts and Nowcast probabilities

All of these values return a colour indicating the level of activity. See the section on colours for more information. Due to the way kp is structured, the colours are their own property. See the example below

Required Parameters

Parameter Type Valid ranges Notes
data String bz Returns Bz details
speed Returns proton speed details
density Returns proton density details
kp Returns current, 1 hour and 4 hour Kp forecasts
threeday Returns 3 day Kp forecast
twentysevenday Returns 27 day Kp forecast
probability Returns data about your best chance of seeing an aurora overhead, within 1000km (i.e. on the horizon) and in the whole world
all Returns all of the above, except for threeday, twentysevenday and probability

Optional Parameters

Parameter Type Valid ranges Notes
lat Float -90 to 90 The latitude of the area you wish to obtain information for. Only required if data is probability
long Float -180 to 180 The longitude of the area you wish to obtain information for. Only required if data is probability

Example

http://api.auroras.live/v1/?type=ace&data=kp
