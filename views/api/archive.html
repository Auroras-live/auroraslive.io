The Archive Module

The Archive module lets you search through previous years' data. Due to the amount of data stored and transmitted, you can only search for three days worth of data in a single call.

If you require more than three days worth of data, you can do multiple calls, but don't do this to download large sets of data. If you need large sets of data (typically more than two weeks), please contact admin@auroras.live for a MySQL dump

Required Parameters

Parameter Type Valid ranges Notes
action String stats Returns information on the archive (size, start and end dates)
search Searches the archives for a particular date
start Date Any valid PHP-compatible date The date to begin retrieving data from. Only required if action is search
end Date Any valid PHP-compatible date The date to end retrieving data from. Only required if action is search

Note

Due to limitations with the database, when tz is set, your start and end dates are converted to UTC, and the results are returned as UTC. This means you can search for an event at 1am local time, and it'll convert that to UTC and retrieve results. It's up to you to convert to your local timezone again. This behaviour may change in the future.

Example 1

http://api.auroras.live/v1/?type=archive&action=stats

Returns

See Example
