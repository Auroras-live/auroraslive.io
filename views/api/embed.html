The Archive Module

The embed module returns an image containing various data, allowing the information to be embedded in a forum signature, website etc. You have the option of embedding current data, or embedding the current weather for a location.

Required Parameters

Parameter Type Valid ranges Notes
image String current Obtains the current space weather conditions
weather Obtains the current weather
lat Float -90 to 90 The latitude of the area you wish to obtain information for. Only required if image is set to weather
long Float -180 to 180 The longitude of the area you wish to obtain information for. Only required if image is set to weather

Example 1

http://api.auroras.live/v1/?type=embed&image=weather&lat=40.7813913&long=-73.976902

Returns

Returns the weather for Manhattan, New York

See Example

Example 2

http://api.auroras.live/v1/?type=embed&image=current

Returns

An image showing the current data (Bz, speed, density and Kp) for a given location. tz aware, so you can display the image in your local time

See Example
