The All Module

The "all" module lets you combine multiple requests into a single call, reducing load times and API calls

Information returned by this module includes:

  • ACE data (speed, density, Bz, current Kp, 1 hour Kp forecast, 4 hour Kp forecast, 3 day Kp forecast)
  • Current and 9 day weather for the specified location
  • Probability data, which includes
    • The probability of seeing an aurora directly overhead
    • The highest probability of seeing an aurora within 9 degrees (approx. 1000km) of your location
    • The location on earth that has the best probability of seeing an aurora overhead

Required Parameters

Parameter Type Valid ranges Notes
lat Float -90 to 90 The latitude of the area you wish to obtain information for
long Float -180 to 180 The longitude of the area you wish to obtain information for

Optional Parameters

Parameter Type Default Valid ranges Notes
ace Boolean true true / false If false, omits ACE data from the results
archive Boolean false true / false If true, includes information about the archive with the results
forecast Boolean true true / false If false, and weather is true, omits 9 day weather forecast from the results
images Boolean false true / false If true, includes a list of images with the results
probability Boolean true true / false If false, omits "nowcast" data from the results
threeday Boolean true true / false If false, omits the 3 day Kp forecast from the results
twentysevenday Boolean false true / false If true, includes the 27 day Kp forecast with the results
weather Boolean true true / false If false, omits all weather from the results

Example

http://api.auroras.live/v1/?type=all&lat=40.7813913&long=-73.976902&forecast=false&threeday=false
