Usage

The Auroras.live API is presently free to use. In order to use this service, you must attribute Auroras.live as the source of the information, but not in such a way that implies that Auroras.live endorses or is the author of the product. Auroras.live is to be written with a capital "A" at the start, except when referring to the URL. Acceptible attribution includes:

Powered by Auroras.live
Information via Auroras.live

Unacceptable attribution includes:

Written by Auroras.live
An Auroras.live app

The attribution should be easily findable by users (e.g. in the footer of the page, or under a Help > About menu, for example) and should always be written with a capital A, except when referring to the URL

To obtain a copy of the logo for embedding purposes (e.g. dynamically generated images), please email admin@auroras.live

Data limits

There are presently no limits on the amount of data you can access, except the archives, which is limited to a total of 3 days of data per call

However we encourage you to use this API responsibly and to cache requests on your end wherever possible. This API is provided for free and costs real money and time to maintain, so your responsible usage is very much appreciated

API limits may be enforced in the future. These will require you to request a key if you wish to use more than a certain number of calls an hour
