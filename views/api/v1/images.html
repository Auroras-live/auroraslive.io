The Images Module

The images module caches and returns images from various webcams, charts, satellite and models.

Required Parameters

Parameter Type Valid ranges Notes
action String list Obtains the a list of images
image String An image as listed by action=list Returns the given image

Example 1

http://api.auroras.live/v1/?type=images&action=list

Returns

Returns a list of images. The cache property tells you how long the image is cached

See Example 

{{ $root.resultsData['imagelist'] }}

Example 2

http://api.auroras.live/v1/?type=images&image=yellowknife

Returns

The aurora camera currently located in Yellowknife, NWT, Canada
