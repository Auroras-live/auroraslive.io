Colours

Many modules will return a colour along with the information. This allows users to know what the returned values actually mean (e.g. green = no activity, red = lots of activity). It's also designed to allow programmers to set colours without having to code in lenghthy switch statements.

If you wish to change the format of the colours, simply append colour= to your request along with hex or rgb to return the colours in that format. If no format specified, colour is the default

Please take note of spelling, as the API uses British English, so colour as opposed to color

Valid Values

Value Notes
colour Default type, returns the colour as a regular colour name (e.g. green)
hex Returns the colour as a hexadecimal string
rgb Returns the colour as a rgb triplet (CSS property)

Example

http://api.auroras.live/v1/?type=ace&data=bz&colour=hex

Returns

Returns the colour in the specified format

See Example 

{{ $root.resultsData['colour'] }}
