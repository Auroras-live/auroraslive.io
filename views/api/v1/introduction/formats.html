Formats (BETA)

By default, data will be returned as JSON. If you wish to return the data as XML, simply append format=xml to your request. Note: this won't work on all modules, as the generated fields contain illegal keys

Valid Values

Value Description Notes
xml Returns the colour as an XML string This may not work with all modules, as some modules return keys that may not be valid XML nodes
jsonp Returns the data as JSONP (example) By default, the callback function name is aurorasLive. To specify a custom callback, use &callback=. Custom callbacks can only contain A-Z, a-z, 0-9, - and _

Example

http://api.auroras.live/v1/?type=ace&data=bz&format=xml

Returns

Returns the colour in the specified format
